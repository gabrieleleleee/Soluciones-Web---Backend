Sistema Digital de Control de Calidad para Producción Artesanal

📖 Descripción del Proyecto
Aplicación web diseñada para digitalizar y centralizar el registro de inspecciones de calidad en pequeñas empresas de manufactura local y artesanal. Este sistema permite rastrear lotes, registrar estándares y generar alertas, alineándose con el ODS 9 (Industria, Innovación e Infraestructura).

 Estructura del Proyecto
El proyecto está dividido en dos aplicaciones principales:
* `/frontend`: Interfaz de usuario (App Web Responsiva) para operarios y administradores.
* `/backend`: API RESTful para la gestión de usuarios, roles, lotes y métricas de calidad.
