# 📋 Documentación Inicial del Proyecto

## 1. Identificación del Problema
Falta de herramientas digitales accesibles para gestionar el control de calidad en la producción artesanal y local. Actualmente se usan métodos manuales (papel), lo que genera pérdida de trazabilidad, imposibilidad de obtener métricas históricas y retrasos en la mejora continua.

## 2. Alternativas de Solución Evaluadas
* *Sistema basado en Hojas de Cálculo (Excel):* Descartado por falta de escalabilidad y baja integridad de datos.
* *App Móvil Nativa:* Descartado por barreras de entrada (requiere hardware específico) y altos costos de desarrollo inicial.
* *App Web Responsiva (Seleccionada):* Permite acceso multiplataforma, actualización centralizada y facilidad para gestionar roles de usuario (Administrador/Operario).

## 3. Caso de Negocio
Se desarrollará una Aplicación Web que centralice el registro de parámetros de calidad. La plataforma validará inspecciones contra estándares predefinidos y generará alertas, facilitando la toma de decisiones y alineando la infraestructura productiva local con los objetivos del *ODS 9 (Industria, Innovación e Infraestructura)*.
